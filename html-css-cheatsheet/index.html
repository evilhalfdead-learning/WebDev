html & css cheatsheet

Table Tags

Tag Name Description
<table> Table Use to define a table
<thead> Table Heading Use to define a table's heading row
<tbody> Table Body Use to define a table's body rows
<tr> Table Row Use to define a table row
<td> Table Data Use to define cells within the rows
<tfoot> Table Footer Use to define the foooter of a table
Use colspan and rowspan attributes to span multiple columns and rows

Common tags

Tag Name Description
<h1>-<h6> Heading 1-6 Use for headings, in decreasing importance. Use h1 once.
<p> Paragraph Use it to define a paragraph
<a> Anchor tag Use to create hyperlinks
<img> Image Use for images and gifs; empty
<ol> Ordered list Use if you want your list items to have numbers
<ul> Unordered list Use if you want your list to have bullets
<li> List Item Put items in your lists
<div> Division Use to make your code have sections

Handy CSS

Rule Name Description
color Color Colors your text
